To-do List

This project was as part of a unit I completed at North Metropolitan TAFE, and involved researching and applying a modern web technology trend. I chose Meteor.js as a framework that I would investigate, learn how to use, and demonstrate it by creating an application.

Meteor is an open-source, isomorphic, full-stack JavaScript platform used to build modern web, desktop, and mobile applications. On the frontend it can be used with any UI library such as React or Vue and the backend, Meteor uses Node with MongoDB being integrated as the default database.

During this project, I have practiced skills in identifying an emerging web technology, implementing a new web technology, and reviewing and finalising the implementation process. I have also developed skills in server-side programming using JavaScript, and the basics of using React as a UI library with TailwindCSS for styling.

This application includes an authentication system where users can register and login to their accounts, and a tasks component where users can create a list and check off tasks as they are completed.

Brick-by-Brick

This project involves creating an online cataloguing application for LEGO collections, and is currently being completed at North Metropolitan TAFE. The client involved, needs a solution to keep track of his LEGO sets in a digital format, and wants to display his collection online.

I have been involved in all aspects of this project, from the initial discussions with the client, to planning and design, and now creating a prototype of the website. Our team has just completed the first half of our development plan; next semester we will be creating a fully functioning web application including a database and authentication system.

This prototype was built using HTML, SCSS, and JavaScript, and the initial mock-ups were designed using Figma.

I have gained experience throughout this process in areas such as:

  • Implementing a structured approach to development using agile techniques
  • Liaising with a client and maintaining frequent progress updates
  • Conceptualizing and developing ideas based on a client’s needs
  • Documenting and outlining an agreement of the scope of a project
  • Developing wireframes, mock-ups, and a prototype from an original design
  • Leading and facilitating a team

Frontend Mentor

Alongside my studies at TAFE, I have been practicing front end development using a variety of online resources including a website called Frontend Mentor. The purpose of this site is to give developers practice in creating webpages based on pre-made designs.

Frontend Mentor offers a range of projects that can be completed using any technology the user would like to practice. I have been solidifying my understanding of HTML, CSS, and JavaScript, as well as using TailwindCSS.

Through these challenges, I have primarily developed my CSS skills, including applying different layout techniques with both CSS Grid and Flexbox, and structured styling using the BEM methodology.

Links to these projects and live sites can be found on my Frontend Mentor profile page, and other repositories that I have been working on in TAFE, and outside of it, can be found on my GitHub.
